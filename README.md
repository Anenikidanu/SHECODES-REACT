# My To-Do App 

Welcome to **My To-Do App**, a simple React application to help you stay organized and productive.  
This project is part of my learning journey in React and front-end development.

---

## Features

- Add new tasks to your to-do list
- Mark tasks as completed
- Delete tasks
- Simple and clean user interface

---

##  Technologies Used

- **React** – Frontend library for building UI
- **JavaScript (ES6+)** – Core logic
- **HTML / CSS** – Structure and styling
